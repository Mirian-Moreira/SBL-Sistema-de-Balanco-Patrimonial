# SBL---Sistema-de-Balan-o-Patrimonial
O SBL - Sistema de Balanço Patrimonial é um CRUD multiplataforma feito em Python com Flet e SQLite. Ele oferece controle completo de ativos, gestão local de fotos, exportação de relatórios em CSV e um dashboard financeiro automático. O projeto foi estruturado e compilado para rodar de forma nativa tanto no Windows (.exe) quanto no Android (.apk).
